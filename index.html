Welcome¶

wt is a small library helping to generate static site or blog or whatever else named static. It tries to be invisible for a blogger and have just three command-line interface commands. At the same time it is not expected to become a "swiss knife" - authoring still requires some user intervention to templates, markup, styles and javascript.

This library requires python 3 and it's recommended to use it within virtual environment. See Quick start for introduction.

Strictly speaking there is no python coding needed to work with the library but it's possible to extend jinja2 templates using selfmade filters and functions.

And please keep in mind that library offers no default css styling or themes.

Happy authoring!

Features¶

  • markdown for content
  • yaml for configuration
  • jinja2 for templates
  • atom for feed
  • two types of content - page and post
  • yaml-formatted front matter for content metadata
  • simple HTTPServer for development
