Getting Started¶

Bootstrap your project¶

Once you have wt installed you can quickly bootstrap your project by running the following command in the terminal:


$ wt init .

You will have directory tree like this one (excluding virtual environment directory):


$ tree -I env
.
├── content
│   ├── pages
│   │   └── foo.md
│   └── posts
│       ├── bar.md
│       └── baz.md
├── static
│   └── css
│       └── style.css
│   └── img
│       └── logo96.png
├── templates
│   ├── atom.xml
│   ├── content.html
│   └── mainpage.html
└── wt.yaml

7 directories, 9 files

Start development server¶

You can start simple HTTPServer server suitable for development by running the following command in the terminal:


$ wt develop
[D 2017-03-31 19:02:55,825 wt.server] Server started at 127.0.0.1:9000...

and navigate to your newly created project in browser. You will see something like this:

For demo purposes there are foo, bar and baz pages available. To stop development server just hit Ctrl-C in the terminal.

That's it - you can start authoring your content straight away.
